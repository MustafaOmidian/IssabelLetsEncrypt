# IssabelLetsEncrypt
get and set Lets Encrypt Certificate on Issabel PBX

# Installation
just run on your Issabel Linux CLI:
```
curl -s -L https://github.com/voipiran/IssabelLetsEncrypt/master/getLetsIssabel.sh | bash
```
